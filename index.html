Quiz Templates

Interactive question components for your developer to integrate

✏ Written Answer

Text input question with visual feedback for correct/incorrect answers. Matches the developer's current design.

🕐 Draggable Clock

Interactive clock with draggable hands. Students set the time by rotating the hour and minute hands.

📦 Drag & Drop Sorting

Drag items into categorized containers (like shapes into jars by number of sides).

🎨 Colour the Blocks

Click blocks to colour them to make a number. Teaches place value with tens and ones.

🔡 Spelling Letters

Drag letters to spell words for each picture. Letters must be placed in order.

🏷 Word Match

Match words to pictures by dragging. Includes distractor words to choose from.

Files for Developer

  • css/base.css Core styles, colors, and layout matching current design
  • css/clock.css Draggable clock component styles
  • css/drag-drop.css Drag and drop component styles
  • js/clock.js Clock interaction logic (DraggableClock class)
  • js/drag-drop.js Drag/drop logic (DragDropZone class)
